queryTopology(word, topology, k=12)
1. Stem input word
2. Compute cosine similarity against all vocabulary using coMatrix rows as vectors
3. Look up PPMI co-neighbors (direct edges, sorted by weight)
4. Compute docAffinity: dot(word_vector, docVector) per manifold
5. Return: { stemmed, wordSims[k], coNeighbors, docAffinity, density, centrality, coherence }
chatTopology(msg, topology, history, opts)
1. Tokenize msg → signal tokens
2. Apply entropyFlux = H(signal) - H(field)  → novelty measure
3. Apply SHADOW perturbation from membrane + attractors
4. NAVIGATE: follow density gradient ρ(x) from signal tokens
   - At each step: choose neighbor maximizing w·centrality·(1-revisit_penalty)
   - Inject hyperfold: secondary path via phase-adjacent terms
   - metaImpression: 4-turn sliding window of dominant path words
5. Apply ANTILOOP: detect recurrence, inject orthogonal term if looping
6. Return: { lines, path, hyperfold, dominantManifold, pressure, entropyFlux, acfScore, metaImpression }

TRI-TICK STATE MACHINE

META-TICK    (`) : backtick-delimited segments
PAREN-TICK   ( ) : parenthetical depth transitions  
BRACKET-TICK [ ] : modifier zone boundaries

State vector: S = (t_m, t_p, t_b) ∈ ℕ³

HUNGRY  ↔  STABLE   governed by:
  session_ticks = Σ(t_m + t_p + t_b) since last RESET
  isHungry = (session_ticks % 2 !== 0)

COLLAPSE RISK:
  base_risk = imbalance(brackets) · 0.4 + chaosPattern · 0.3 + depth_overflow · 0.3
  
INVERSION EVENT (stabilization):
  Triggered when: bracket-tick catches collapse risk ≥ 0.8
  Effect: collapseRisk → 0, inversionTriggered = true
  Semantics: [BRACKET] absorbs (PAREN) pressure — structural damping

RESET:
  Triggered by: "!!!!ONE!!!!ONE!" token sequence
  Effect: session_ticks = 0, isHungry = false, collapseRisk = 0
